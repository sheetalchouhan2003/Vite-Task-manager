# Task Manager App

This is a simple Task Manager application built using **React (Vite)**.

---

## Features

- Add new tasks
- Edit and delete tasks
- Drag and drop tasks between boards
- Filter tasks by priority and status
- Sort tasks by date
- Data saved using localStorage

---

## Technologies Used

- React (Vite)
- JavaScript
- Tailwind CSS
- @hello-pangea/dnd
- localStorage

---

## Project Setup

1. Install dependencies
```bash
npm install

2. Run the project

npm run dev

Usage

Click Add Task to create a task

Click on a task to edit or delete

Drag tasks between To-Do, In-Progress and Completed boards

Use filters and sorting to manage tasks

Data remains after page refresh
